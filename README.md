# India General Election Result Analysis - 2024 (Power BI)

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze the **2024 Indian General Election Results**. The dashboard provides comprehensive insights into election outcomes, political alliances, state-wise performance, constituency-level results, and detailed candidate statistics using interactive visuals and maps.

---

## 🚀 Project Objectives

- Analyze the 2024 Lok Sabha Election Results.
- Compare NDA, I.N.D.I.A., and Other Alliances.
- Explore state-wise political landscape.
- Analyze constituency-level voting patterns.
- Identify winning candidates and vote margins.
- Provide an interactive dashboard for better decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX
- Bing Maps
